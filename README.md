# 📚 电子信息类 · 期末复习资料传承计划

## 为什么有这个仓库？

我是 **23 通信陈****，正在备考 985。

如果成功上岸，我会是本学院近十年来第一个考上 985 的学生。这一路走来，我清楚普通本科的资源有多匮乏——没有学长带路、没有现成的复习资料、每一步都得自己踩坑。

这个仓库就是我想留下的东西。把我整理的期末资料按课程分类归档，一届传一届，让学弟学妹至少在面对期末时，不用再从零开始。

**它应该比我走得更远。**

## 这是什么

按课程分类整理的 **电子信息类** 期末复习资料库，覆盖：

- 📄 **期末复习题 & 答案**（含 LaTeX 排版 PDF）
- 📝 **重点总结 & 知识提纲**
- 💡 **真题回忆 & 题型归纳**
- 📓 **个人笔记（后续补充）**

> 本学院电子信息大类下分 **电子信息工程** 和 **通信工程** 两个专业，核心课程高度重叠，所以放在一起维护，覆盖面更广、复用率更高。
>
> **无论你是哪个专业，只要课程对得上，就能用。**

## 当前维护边界

本仓库优先提交**本人整理、可长期维护、版权边界清晰**的资料。

### 已确认可推送
- 通信原理：`期末考试/` 下的期末复习考纲、复习题目 PDF 与 LaTeX 源码。
- 高频电子线路：`期末考试/` 下的复习题目、考纲、配图、LaTeX 源码，以及本人完成的课程设计报告、调试文档和相关说明。
- 项目维护文件：README、AGENTS/CLAUDE 规则、必要的 `.gitignore` 和自定义 Agent/Skill 配置。

### 待整理后再收录
- 移动通信：期末复习 PDF/LaTeX 已在本地，但需要确认来源、整理说明后再提交。
- 数字信号处理：复习大纲、课后题整理和公开课回答可作为候选，需补充来源说明。
- 数据结构：当前主要是课程设计材料，不是期末复习资料；建议另做课程说明或筛选后提交。
- 电磁场与电磁波：本地有期末作业/论文类材料，建议整理成知识提纲或经验说明后提交。

### 不直接上传
- 完整教材、教辅、课后答案、商业课程讲义、付费资料、来源不明的扫描 PDF。
- 教材拆分章节、书籍目录、考研辅导讲义、习题册、教师未明确允许传播的课件或试卷。
- LaTeX 编译中间文件、日志、上传缓存、本地工具链和依赖缓存。

## 🤝 如何参与

这个仓库能真正产生价值的时刻，是当它不再只属于我一个人。

### 提交资料
- **Fork → 添加/更新资料 → 提交 PR**
- 可以是真题回忆、笔记补充、答案纠错、LaTeX 重排
- PR 通过后你的名字会出现在下方贡献者列表中

提交资料时建议注明：

```md
资料类型：
适用课程：
适用年份/学期：
资料来源：本人整理 / 考后回忆 / 公开资料链接 / 教材章节索引
是否包含他人内容：
贡献者：
```

欢迎提交：
- 自己整理的复习提纲、知识点总结、公式表、错题归纳。
- 自己完成并校对过的习题解析、证明过程、计算过程。
- 考后凭记忆整理的真题回忆、题型分布、重点章节说明。
- 对已有资料的纠错、补充、排版优化、LaTeX 重排。
- 公开来源资料的索引、链接、阅读建议和章节对应关系。

不建议提交：
- 未经授权复制的教材、课后答案、教辅扫描件。
- 教师未公开发布或明确禁止传播的课件、试卷、答案。
- 来源不明、无法核验版权边界的资料。
- 直接从其他平台搬运且未注明来源的内容。
- 可能被用于作弊、泄题或规避考试纪律的内容。

### 反馈问题
- **提交 Issue** 报告资料错误、缺失课程或提出建议

### 传承约定
如果你是后来的学弟学妹，这个仓库对你有帮助的话——
- 期末考完后，把你记得的 **真题回忆** 或 **整理的新笔记** 提交 PR
- **让这份资料一届一届传下去**
