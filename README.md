# 📚 电子信息类 · 期末复习资料传承计划

## 为什么有这个仓库？

我是 **23 通信陈文斗**，正在备考 985。

如果成功上岸，我会是本学院近十年来第一个考上 985 的学生。这一路走来，我清楚普通本科的资源有多匮乏——没有学长带路、没有现成的复习资料、每一步都得自己踩坑。

这个仓库就是我想留下的东西。把我整理的期末资料按课程分类归档，一届传一届，让学弟学妹至少在面对期末时，不用再从零开始。

**它应该比我走得更远。**

## 这是什么

按课程分类整理的 **电子信息类** 期末复习资料库，覆盖：

- 📄 **期末复习题 & 答案**（含 LaTeX 排版 PDF）
- 📝 **重点总结 & 知识提纲**
- 💡 **真题回忆 & 题型归纳**
- 📓 **个人笔记（后续补充）**

> 本学院电子信息大类下分 **电子信息工程** 和 **通信工程** 两个专业，核心课程高度重叠，所以放在一起维护，覆盖面更广、复用率更高。
>
> **无论你是哪个专业，只要课程对得上，就能用。**

## 🤝 如何参与

这个仓库能真正产生价值的时刻，是当它不再只属于我一个人。

### 提交资料
- **Fork → 添加/更新资料 → 提交 PR**
- 可以是真题回忆、笔记补充、答案纠错、LaTeX 重排
- PR 通过后你的名字会出现在下方贡献者列表中

### 反馈问题
- **提交 Issue** 报告资料错误、缺失课程或提出建议

### 传承约定
如果你是后来的学弟学妹，这个仓库对你有帮助的话——
- 期末考完后，把你记得的 **真题回忆** 或 **整理的新笔记** 提交 PR
- **让这份资料一届一届传下去**
